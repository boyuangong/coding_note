\section{Basic questions}
\subsection{find missing number}
Find missing number problem can have many different formulas. 

First, let's look at find one missing number.
\begin{problem}
Given an array containing n distinct numbers taken from 0, 1, 2, ..., n, find the one that is missing from the array.

For example,
Given nums = [0, 1, 3] return 2.
\end{problem}

In this problem we need to use XOR($\oplus$) to prevent the overflow.
XOR have some properties:
\begin{enumerate}
\item For bit:
\begin{enumerate}
\item $0\oplus0 = 0$
\item $0\oplus1 = 1$
\item $1\oplus0 = 1$
\item $1\oplus1 = 0$
\end{enumerate}
\item For numbers
\begin{enumerate}
\item $x\oplus x = 0$
\item $x\oplus 0 = x$ actually this is the special case for $x\oplus y = z$, then
$x\oplus z = y$
\item $x\oplus y\oplus z = x\oplus z\oplus y = y\oplus x\oplus z$
\end{enumerate}
\end{enumerate}

Thus we only need to do is xor all the numbers in the array with the array of the index. Thus we can find the missing number.
